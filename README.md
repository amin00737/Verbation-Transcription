🗣️ Verbation-Transcription

A speech-to-text transcription workflow using WhisperX (Whisper V2/V3) for high-quality audio transcription with optional speaker diarization and word-level alignment. This project contains an example Jupyter notebook demonstrating these capabilities and Python script templates.

No repository description is currently provided in the project — this README fills that gap.

📌 Project Overview

The goal of Verbation-Transcription is to provide a reproducible workflow for converting audio recordings into time-aligned transcripts with optional speaker labels. It uses WhisperX, an enhanced transcription framework built on OpenAI’s Whisper models extended with alignment and diarization support.

This repository includes:

A Jupyter Notebook (Whisper V2 V3.ipynb) showcasing transcription and alignment.

A Python script template for extending to other audio pipelines.

Languages used:
📝 Jupyter Notebook — 97%
🐍 Python — 3%

🛠️ Features

✔️ Audio file transcription using WhisperX
✔️ Word-level alignment (timestamps & segmentation)
✔️ Optional speaker diarization
✔️ Notebook example demonstrating typical usage
✔️ Extensible for custom workflows
